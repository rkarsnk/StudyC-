## 1.2 まずは画面に表示
### ヘッダインクルード 
`#include` で始まる文。
```
#include <iostream>
```
入出力に関する`<iostream>`や文字列に関する`<string>`などのヘッダが提供されている。
この記述を削除すると、コンパイルに失敗する。


### 名前空間
`using`で始まる。以下の例では、標準ライブラリの名前空間stdを指定している。
```
using namespace std;
```
標準ライブラリの利用に必要な決まり文句。省略する場合、`cout`の記述を`std::cout`とする必要がある。


### 標準出力の書き方
`cout`を利用する。`cout`は標準出力ストリーム、`<<`は挿入子。
```
    cout << "初めてのC++プログラム。\n";
    cout << "画面に出力しています。\n";
```

```
    cout << "初めてのC++プログラム。\n画面に出力しています。\n";
```
2行の出力を1行で記述することも可能。

### ストリームへの連続した出力
挿入子`<<`を連続して利用することが可能。
例では、`\a`で警報出力後に「はじめまして。こんにちは。」が出力される。

### 自由記述形式
ソースコードが見づらくなるので、原則禁止。


### 文字列リテラル
文字列リテラル`"ABCDEFGHIJKLMNOPQRSTUVWXYZ"`と`"abcdefghijklmnopqrstuvwxyz\n"`が連結され、1個の文字列リテラルとなる。

## 1.3 変数
### 整数リテラルの演算結果の出力
```
./list0106 
18 足す 63 は 81です。
```
`list0105.cpp`では、`18 + 63`が演算され出力される。
`18`や`63`は整数リテラル、`"18 足す 63 は"`と`"です。\n"`は文字列リテラルという。

### 変数の初期化
```
 ./list0107
xの値は63です。
yの値は18です。
合計は81です。
平均は40です。
```
`list0107.cpp`では、
```
int x;
int y;

x = 63;
y = 81;
```
のように、変数`x`と`y`の宣言後に、代入演算子で`x`に`63`、`y`に`81`を代入している。
```
./list0108 
xの値は0です。
yの値は0です。
合計は0です。
平均は0です。
```
`list0108.cpp`では、`x`と`y`の値は不定値となる。

`list0109.cpp`では、
```
int x = 63;
int y = 18;
```
と初期化している。

以下の記述形式による初期化が可能。
```
int x(64);
int x{63};      //C++11以降
int x={63};     //C++11以降では
```

## 1.4 キーボードからの入力
`cin`を利用する。`cin`は標準入力ストリーム、`>>`は抽出子。
```
cout << "xの値:";
cin >> x;
    
cout << "yの値:";
cin >> y;
```
`list0110.cpp`では、
```
xの値:7
yの値:5
```
と入力することで、`x`と`y`に値を読み込む。

`x + y`の`x`,`y`はオペランド、`+`は算術演算子。`+`、`-`、`*`、`/`、`%`は全てオペランドを2つ持つ二項演算子である。
### 値の連続した読込み
`list0111.cpp`では、値を連続して読み込む。
```
 ./list0111 
xとyを加減乗除します。
xとyの値:7 5     ←空白挿入
x + y = 12
x - y = 2
x * y = 35
x / y = 1
x % y = 2

./list0111 
xとyを加減乗除します。
xとyの値:7       ←改行
5
x + y = 12
x - y = 2
x * y = 35
x / y = 1
x % y = 2
```
抽出子`>>`を使った入力では、空白・タブ・改行などの空白文字が読み飛ばされる。

### 単項演算子
`+a`は`a`そのものの値を生成。`-a`は`a`の符号を反転した値を生成。
```
./list0112 
整数値:12
12の符号反転した値は-12です。

./list0112 
整数値:-45
-45の符号反転した値は45です。
```

### 実数値の読込み
`int`は整数型で小数部をもつ実数を扱えないので、`double`という型を利用する。
原則として、整数は`int`型、実数は`dobule`型で表現する。
```
double x;
double y;
```

```
./list0113 
xとyを加減乗除します。
xの値:7.5
yの値:5.25
x + y = 12.75
x - y = 2.25
x * y = 39.375
x / y = 1.42857
```
ちなみに、`x % y`は、このプログラムでは以下のようにコンパイルエラーとなる。
```
list0113.cpp:21:29: error: invalid operands to binary expression ('double' and 'double')
    cout << "x % y = " << x % y << "\n";
                          ~ ^ ~
1 error generated.
```

### 定値オブジェクト
```
const double PI = 3.1416;
```
定値オブジェクトの変数名は大文字にする。そうすることで、他の変数と見分けやすくなる。

また定値オブジェクトは、宣言時に初期化子が必須。以下のコードはコンパイルエラーとなる。
```
const double PI;
PI = 3.1416;
```

### 乱数の生成
```
#include <ctime>                //C言語のtime.hと同じ
#include <cstdlib>              //C言語のstdlib.hと同じ
#include <iostream>

using namespace std;

int main(){
    srand(time(NULL));          //現在の時刻を乱数の種に使用

    int lucky = rand() % 10;    //0～9の乱数

    cout << "今日のラッキーナンバは" << lucky <<"です。\n";
}
```
rand関数で生成される乱数は疑似乱数である。

### 文字の読込み
1文字だけなら
```
char c
```

### 文字列の読込み
`cin`利用して文字列を読み込む場合は、
```
#include <string>              //文字列を扱うヘッダライブラリ
#include <iostream>

using namespace std;

int main(){
    string name;

    cout << "お名前は:";
    cin >> name;

    cout << "おはようございます、"<< name << "さん。\n";
}
```
空白を含んだ文字列を読み込む場合は、
```
cin >> name;
```
を
```
getline(cin, name);
```
とする。

文字列を初期化する場合は、
```
string s1 = "ABC";
string s2 = "XYZ";
```
文字列を変数に代入する場合は、
```
string s1 = "ABC";
s1 = "CIA";             //代入
```
のようにする。
